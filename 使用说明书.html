MK-tool 使用说明书

这份文档面向日常使用，不是开发说明。内容重点是：怎么配置、怎么下载、默认文件怎么生效、目录怎么命名、以及常见问题怎么排查。

适用浏览器：Chrome / Edge 快捷键：Alt + Q 当前版本：1.1.4

一、扩展介绍

MK-tool 是一个浏览器扩展，主要用于门店资料下载、三图资料整理、字段提取，以及钉钉表单同步。

一键提取并下载

提取当前页面里的图片、附件和相关资料，并按门店目录自动整理下载。

三图资料下载

下载效果图、施工图、三图确认相关附件，并归到对应门店目录。

字段提取

识别门店编码、上匠门店简称、加盟商信息、地址、面积、完成时间等字段。

同步钉钉

将当前页面提取到的字段自动填入你配置好的钉钉表单，并支持快速打开表格地址。

二、安装方法

  1. 打开扩展管理页：Chrome 用 chrome://extensions，Edge 用 edge://extensions。
  2. 打开“开发者模式”。
  3. 点击“加载已解压的扩展程序”。
  4. 选择当前项目文件夹 MK_tools。

提示： 如果你后面更新了扩展代码，需要回到扩展管理页点击一次“重新加载”，再刷新目标网页。

三、第一次使用前设置

建议你先打开扩展设置页，再按下面顺序完成配置。

3.1 默认文件

默认文件是一些会跟门店资料一起落地的预设文件，适合放模板、清单、说明文件、设备清单等。

你可以设置什么

  • 默认文件存放文件夹名
  • 默认文件列表

怎么添加默认文件

  • 点击“添加”选择文件
  • 把文件拖进默认文件区域
  • 使用 Ctrl + V 粘贴文件

什么时候默认文件才会生效

  • 默认文件保存在插件设置里，不是直接引用你电脑上的原始文件。
  • 只有在页面悬浮面板里打开“配置默认文件”开关后，再点击“一键提取并下载”，这些默认文件才会一起复制。
  • 点击“三图资料下载”时，不会带出默认文件。
  • “配置默认文件”开关会记住上一次状态。

默认文件会复制到哪里

  • 如果不填“默认文件存放文件夹”，默认文件会直接放进门店主目录。
  • 如果填写了“默认文件存放文件夹”，默认文件会放进这个子文件夹。
  • 这里填写的是门店目录下的子文件夹名称，不是电脑里的完整路径。
未填写文件夹名：
序号--门店编码上匠门店简称/默认文件A

填写了“默认文件”：
序号--门店编码上匠门店简称/默认文件/默认文件A

文件名替换规则

写法 实际替换结果
XXXXXX上匠简称 替换为“门店编码 + 上匠门店简称”
xxxx.xx.xx 替换为当天日期，格式为 YYYY.MM.DD

文件内容替换规则

  • 文本类默认文件会自动替换字段占位符。
  • Excel 默认文件会尝试替换单元格中的字段内容，当前以 .xlsx 支持最稳定。
  • 图片、PDF、压缩包这类普通二进制文件会按原样复制。

当前支持的常见占位符包括：

  • {{门店编码}}
  • {{门店编号}}
  • {{上匠门店简称}}
  • {{加盟商名称}}
  • {{加盟商联系方式}}
  • {{完整地址}}

注意： 这些内容替换依赖当前页面字段提取成功。如果页面没有成功提取出字段，占位内容通常会保持原样。

“更新设备清单”按钮规则

  • 它主要用于更新默认文件中的 Excel 设备清单。
  • 如果默认文件里已经有一个 .xlsx 或 .xls 文件，这个按钮会用新文件替换掉第一个已有的 Excel 文件。
  • 如果还没有 Excel 设备清单，则会新增一个。
  • 这个按钮不是追加第二份设备清单，而是优先做“替换”。

3.2 序号门店映射

这个区域用来维护：

  • 月份
  • 序号
  • 门店编码
  • 上匠门店简称

支持：

  • 手动编辑
  • 导入表格
  • 导出表格
  • 清空数据

当前规则： 同一个门店编码，只会在第一次下载时写入映射；以后再次下载，会直接复用原来的序号、月份和门店简称；后续下载不会因为跨月而自动改成新月份；如果导入表格里已经有该门店编码，后面下载时不会重复新增，也不会覆盖原记录。

3.3 钉钉信息设置

钉钉地址配置区域放在设置页的最后面。

  • 钉钉表单地址：用于“同步表单”，没有默认地址，必须手动填写
  • 钉钉表格地址：用于“打开表单”，没有默认地址，必须手动填写

注意： 现在不再提供默认钉钉地址；如果不填写，对应按钮将无法使用。

四、页面里怎么使用

进入目标页面后，插件会注入一个悬浮面板。

快捷键： Alt + Q，可用于打开或关闭悬浮面板。

4.1 悬浮面板按钮说明

提取字段

自动识别当前页面中的门店信息，并显示在面板下方。常见字段包括门店编码、上匠门店简称、加盟商信息、完整地址、项目类型、面积、完成时间、状态等。

一键提取并下载

下载当前页面中可识别的图片和文件，并按门店信息生成对应目录。如果同时打开了“配置默认文件”开关，这一步还会把默认文件一起复制到门店目录。

三图资料下载

下载页面中的效果图、施工图、三图确认相关附件，并归到对应门店目录下的“三图资料”目录。

同步表单 / 打开表单

“同步表单”会用当前提取到的字段自动填入钉钉表单；“打开表单”只负责跳转到你在设置页配置好的钉钉表格地址。

推荐使用流程

  1. 先打开目标页面。
  2. 按 Alt + Q 打开悬浮面板。
  3. 先检查字段是否提取正确，特别是门店编码。
  4. 如需下载现场资料，点击“一键提取并下载”。
  5. 如需下载三图资料，点击“三图资料下载”。
  6. 如需同步钉钉，点击“同步表单”。
  7. 如只需跳转查看钉钉地址，点击“打开表单”。

五、下载目录规则

插件会根据门店信息自动创建下载目录。常见命名规则为：

序号--门店编码上匠门店简称

在这个门店目录下，再放入不同资料，例如：

  • 现场资料
  • 三图资料
  • 默认文件

如果你在设置页中填写了“默认文件存放文件夹”，默认文件会进入这个子目录；否则直接放在门店主目录下。

六、常见问题

1. 按钮点了没反应

  • 检查当前页面是否已经刷新到最新。
  • 检查扩展是否刚更新过。
  • 检查设置页中的钉钉地址是否已填写。

如果你刚改过代码或重新加载过扩展，请先在扩展管理页点击“重新加载”，再回到目标网页刷新页面。

2. 下载目录不对

  • 检查当前页面是否成功识别到门店编码。
  • 检查序号门店映射表中是否已经存在该门店编码。

如果该门店编码已经存在，插件会优先复用旧映射，不会自动按新月份覆盖。

3. 三图资料没有进对应目录

  • 门店编码未识别成功。
  • 映射表中已有旧记录并被复用。
  • 浏览器下载路径设置影响了最终保存位置。

4. 默认文件没有复制进去

  • 检查设置页里是否已经真的添加了默认文件。
  • 检查页面悬浮面板里的“配置默认文件”开关是否已打开。
  • 确认你点的是“一键提取并下载”，而不是“三图资料下载”。
  • 检查当前页面字段是否提取成功。

5. 同步表单失败

  • 检查钉钉表单地址是否填写正确。
  • 检查当前页面字段是否提取成功。
  • 检查表单页面结构是否发生变化。
