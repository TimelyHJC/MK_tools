# MK-tool 使用说明（中文版）

`MK-tool` 是一个浏览器扩展，主要用于门店资料下载、三图资料整理、字段提取，以及钉钉表单同步。

本文档以“日常使用” 为主，不是开发说明。

## 这个插件能做什么

- 一键提取当前页面里的图片和附件并下载
- 下载“三图资料”
- 自动识别页面中的门店信息
- 把识别到的字段同步到钉钉表单
- 打开预先配置好的钉钉表格地址
- 管理默认文件
- 维护门店编码与序号的对应关系

## 适用浏览器

- Chrome
- Edge
- 其他 Chromium 内核浏览器

## 安装方法

1. 打开扩展管理页
   - Chrome：`chrome://extensions`
   - Edge：`edge://extensions`
2. 打开“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前项目文件夹 `MK_tools`

## 第一次使用前要做的设置

打开扩展设置页后，建议按下面顺序配置。

### 1. 默认文件

可以上传一些固定会跟门店资料一起存放的文件。

用途：

- 新建门店资料目录时自动带上这些文件
- 适合放模板、清单、说明文件等

你可以设置：

- 默认文件存放文件夹名
- 默认文件列表

添加方式：

- 点击“添加”选择文件
- 把文件拖进默认文件区域
- 使用 `Ctrl + V` 粘贴文件

规则说明：

- 默认文件是保存在插件设置里的，不是直接引用你电脑上的原始文件
- 只有在页面悬浮面板里打开“配置默认文件”开关后，再点击“一键提取并下载”，这些默认文件才会一起复制
- 点击“三图资料下载”时，不会带出默认文件
- 这个“配置默认文件”开关会记住上一次状态

复制位置：

- 如果不填“默认文件存放文件夹”，默认文件会直接放进门店主目录
- 如果填写了“默认文件存放文件夹”，默认文件会放进这个子文件夹
- 这里填写的是门店目录下的子文件夹名称，不是电脑里的完整路径

实际效果示例：

```text
未填写文件夹名：
序号--门店编码上匠门店简称/默认文件A

填写了“默认文件”：
序号--门店编码上匠门店简称/默认文件/默认文件A
```

文件名替换规则：

- 文件名中的 `XXXXXX上匠简称` 会替换为“门店编码 + 上匠门店简称”
- 文件名中的 `xxxx.xx.xx` 会替换为当天日期，格式为 `YYYY.MM.DD`

文件内容替换规则：

- 文本类默认文件会自动替换字段占位符
- Excel 默认文件会尝试替换单元格中的字段内容，当前以 `.xlsx` 支持最稳定
- 图片、PDF、压缩包这类普通二进制文件会按原样复制

当前支持的常见占位符包括：

- `{{门店编码}}`
- `{{门店编号}}`
- `{{上匠门店简称}}`
- `{{加盟商名称}}`
- `{{加盟商联系方式}}`
- `{{完整地址}}`

注意：

- 这些内容替换依赖当前页面字段提取成功
- 如果页面没有成功提取出字段，占位内容通常会保持原样

设备清单规则：

- “更新设备清单”主要用于更新默认文件中的 Excel 设备清单
- 如果默认文件里已经有一个 `.xlsx` 或 `.xls` 文件，这个按钮会用新文件替换掉第一个已有的 Excel 文件
- 如果还没有 Excel 设备清单，则会新增一个
- 这个按钮不是追加第二份设备清单，而是优先做“替换”

### 2. 序号门店映射

这个区域用来维护：

- 月份
- 序号
- 门店编码
- 上匠门店简称

支持：

- 手动编辑
- 导入表格
- 导出表格
- 清空数据

当前规则：

- 同一个门店编码，只会在第一次下载时写入映射
- 以后再次下载，会直接复用原来的序号、月份和门店简称
- 后续下载不会因为跨月而自动改成新月份
- 如果导入表格里已经有该门店编码，后面下载时不会重复新增，也不会覆盖原记录

### 3. 钉钉信息设置

钉钉地址配置区域放在设置页的最后面。

需要配置两项：

- 钉钉表单地址：用于“同步表单”，没有默认地址，必须手动填写
- 钉钉表格地址：用于“打开表单”，没有默认地址，必须手动填写

如果不配置，页面里的相关按钮将无法正常使用。

## 页面里怎么使用

进入目标页面后，插件会注入一个悬浮面板。

快捷键：

```text
Alt + Q
```

可用于打开或关闭悬浮面板。

## 悬浮面板按钮说明

### 提取字段

插件会尽量自动识别当前页面中的门店信息，并显示在面板下方。

常见提取字段包括：

- 门店编码
- 上匠门店简称
- 加盟商名称
- 加盟商联系方式
- 预估流水
- 外卖占比
- 外卖流水
- 完整地址
- 项目类型
- 省份
- 拓展顾问
- 接案时间
- 店宽
- 装修面积
- 完成时间
- 状态

如果没有提取到门店编码，部分下载和同步功能会受到影响。

### 一键提取并下载

这个按钮用于下载当前页面中可识别的图片和文件。

适合场景：

- 勘场资料下载
- 页面附件归档
- 按门店整理现场资料

下载时会优先结合：

- 当前门店编码
- 上匠门店简称
- 已存在的门店映射
- 当前序号

来生成对应目录。

如果你同时打开了“配置默认文件”开关，插件还会在这一步把设置页中的默认文件一起复制到门店目录。

### 三图资料下载

这个按钮用于下载页面中的“三图相关文件”，例如：

- 效果图
- 施工图
- 三图确认相关附件

下载结果会放到对应门店目录下的“三图资料”相关目录中。

### 同步表单

这个按钮会：

1. 使用当前页面已提取到的字段
2. 打开你配置好的钉钉表单地址
3. 自动尝试填入字段

使用前提：

- 已成功提取到页面字段
- 已在设置页填写“钉钉表单地址”
- 不再使用默认钉钉表单地址

### 打开表单

这个按钮只负责打开设置页里配置的“钉钉表格地址”，方便你快速跳转。

使用前提：

- 已在设置页填写“钉钉表格地址”
- 不再使用默认钉钉表格地址

## 下载目录规则

插件会根据门店信息自动创建下载目录。

常见命名规则为：

```text
序号--门店编码上匠门店简称
```

在这个门店目录下，再放入不同资料，例如：

- 现场资料
- 三图资料
- 默认文件

如果你在设置页中填写了“默认文件存放文件夹”，默认文件会进入这个子目录；否则直接放在门店主目录下。

## 门店映射规则

这是当前非常重要的一条规则：

- 首次下载某个门店编码时，才会写入映射
- 之后再次下载同一个门店编码，会复用原来的记录
- 不会因为到了新月份就自动改月份
- 不会因为再次下载就覆盖原来的门店简称
- 如果列表里已经有同门店编码，就不再新增

也就是说：

如果某门店最早是在 3 月导入或下载建立的记录，后面 4 月、5 月再次下载时，仍然保持原来的月份和序号，不会被自动改掉。

## 推荐使用流程

日常使用建议按下面顺序操作：

1. 先打开目标页面
2. 按 `Alt + Q` 打开悬浮面板
3. 先检查字段是否提取正确
4. 如需下载现场资料，点击“一键提取并下载”
5. 如需下载三图资料，点击“三图资料下载”
6. 如需同步钉钉，点击“同步表单”
7. 如只需跳转查看钉钉地址，点击“打开表单”

## 常见问题

### 1. 按钮点了没反应

先检查：

- 当前页面是否已经刷新到最新
- 扩展是否刚更新过
- 设置页中的钉钉地址是否已填写

如果你刚改过代码或重新加载过扩展，请：

1. 在扩展管理页点击“重新加载”
2. 回到目标网页刷新页面
3. 再重新操作

### 2. 下载目录不对

先检查：

- 当前页面是否成功识别到门店编码
- 序号门店映射表中是否已经存在该门店编码

如果该门店编码已经存在，插件会优先复用旧映射，不会自动按新月份覆盖。

### 3. 三图资料没有进对应目录

通常与以下情况有关：

- 门店编码未识别成功
- 映射表中已有旧记录并被复用
- 浏览器下载路径设置影响了最终保存位置

建议先检查当前门店编码和映射表内容是否一致。

### 4. 同步表单失败

先检查：

- 钉钉表单地址是否已填写且填写正确
- 当前页面字段是否提取成功
- 表单页面结构是否发生变化

如需进一步查看表单同步设计，可参考：

- [表单同步优化说明](./docs/%E8%A1%A8%E5%8D%95%E5%90%8C%E6%AD%A5%E4%BC%98%E5%8C%96%E8%AF%B4%E6%98%8E.md)

### 5. 默认文件没有复制进去

先检查：

- 设置页里是否已经真的添加了默认文件
- 页面悬浮面板里的“配置默认文件”开关是否已打开
- 你点的是不是“一键提取并下载”而不是“三图资料下载”
- 当前页面字段是否提取成功

如果以上都正常，再重新执行一次下载操作。
